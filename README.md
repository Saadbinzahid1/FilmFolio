# 🎬 FilmFolio – Your Personal Movie Portfolio

FilmFolio is a modern React application that allows users to search for movies, track what they’ve watched, and rate them out of 10.

It acts as a personal movie portfolio where you can organize and evaluate your film journey.

---

## 🌐 Live Preview

[Live Demo](https://filmfolioreact.netlify.app)

---

## 🚀 Features

- 🔎 Search movies in real-time
- ➕ Add movies to your watched list
- ⭐ Rate movies out of 10
- 🗑️ Remove movies from list
- 📊 View average ratings
- ⚡ Instant UI updates with React state
- 📱 Responsive and clean interface

---

## 🛠️ Built With

- React (Functional Components)
- useState & useEffect Hooks
- Modern JavaScript (ES6+)
- CSS3
- Movie API (e.g., OMDb API)

---

- Managing application state

- Controlled inputs
- Calculating derived state (average ratings)
